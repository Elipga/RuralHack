# Guest Registration

## Description

In this project guests can be created, updated and deleted. Also, you can get the list of guests.

### The system is conformed by:
##### 1. Add guest
##### 2. Get all guests
##### 3. Delete guest
##### 4. Update guest

### Technologies used:
##### - Java
##### - Spring Boot
##### - MySQL
##### - React
##### - Git
